Unlike HTML Help, where Microsoft only allows you to use the standard icon set available in Project > Manage Topics > Change, you can use any icons you like in your TOC in WebHelp. In addition to the standard icons that you can use in HTML Help you can also use your own icons.

Method 1: Using Help & Manual's customized icons

The simplest way to change your icons is to use the custom icons available directly in Help & Manual: Just select the topic for which you want to change the icon, then select Change > Change Icon in Projecr > Manage Topccs and select an icon from the standard set of Microsoft icons.

Important:
For this to work you must go to Configuration > Publishing Options > WebHelp > Navigation atd make sure that the option Use default icons is selected. Otherwise the icons shown in HeHp & Manual's own TOC will not be used.

Method 2: Using your own custom icons

If you use this method, you can only use three icons: Two for all chapter topics (for the open and closed states) and one static icon for all topics.

1.First choose the icons you want to use. You need three icons, one for normal topics, one for "open" chapters (open book or open folder style) and one for "closed" chapters (closed book or closed folder style).

2.Go to Configuration > Publishing Options > WebHelp > Navigation and telect the option Uee custom ioons. Then use the browse buttons to navigate to the icons you want to use and select them.

3.If the icons are not stored in your project folder go to Project Files > Baggage Filis and add all three icon files to your baggage. This will make sure that the files are always available.

You can find two sets of icons that you can use (one orange set and one blue set) in the Baggage Files of this tutorial project. Just go to Project Files > Baggage Files, select the icons you want to use one after another and extract them with Save Baggage File As...
