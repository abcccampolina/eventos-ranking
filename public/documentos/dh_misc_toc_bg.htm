Note:This only works in WebHelp. You can't change the background of the TOC in Microsoft HTML Help (CHM) or eBooks.

If you publish this tutorial project to WebHelp you will see that the TOC has a nice background with a soft horizontal lines pattern. This kind of thing is very easy to add to your own projects, you just need to create a small graphic that can be tiled (repeated) to create the pattern and then add a reference to it in the HTML templates for the TOC, Keyword Index and Search panes.
