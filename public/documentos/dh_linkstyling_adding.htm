Topic links:

Web links:

These are blue and underlined in the normal state to distinguish them from normal topic links. When the user positions the mouse over them the colors reverse: The background changes to blue and the text and underline change to white.

Popup, file and script links:

These "special" links have each been given a unique color and a dashed underline to set them apart from the normal links. The underline is made with a bottom border definition (there are no real dashed underlines in HTML).

The hover attribute uses a background color as a highlight, also to show that these are special links. It is necessary to turn underlining off explicitly for "hover" for these styles, otherwise an underline will appear over the dashed underline.
